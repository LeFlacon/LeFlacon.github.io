A. Birthday

http://codeforces.com/contest/1068/problem/A

题意：有n种不同硬币，寿星已经有k种硬币，他有m个朋友，每个朋友会给他送硬币，对于每个朋友来说有三个要求：1.所有朋友送的硬币数相同 2.所有要送的硬币种类不同 3.最终寿星要获得至少l种新硬币，求每个朋友需要送的最小硬币数

就是求满足下式的最小ans，没有的话就是-1

B. LCM（gcd）

http://codeforces.com/contest/1068/problem/B

题意：给定b，对所有a[1,1e18]，求lcm(a,b)/a可能的取值个数

如下化简，因为a可以取所有值，所以最后求的就是b的因子个数

C. Colored Rooks（构造）

http://codeforces.com/contest/1068/problem/C

题意：

n个颜色，m种颜色关系，在1e9*1e9的图中摆放这些颜色

如果两个颜色有关系就要放在同一行同一列，否则就不行，相连就是指在同一行同一列

要求满足：1）每种颜色必须用一次，2）同色必须相连，3）有关系的才能相连，没关系的不能相连

构造思路：

每个颜色放一行，如果有颜色和它有关系，就依次加在那一列上，反正是1e9的棋盘，肯定能放下

比如1和24有关，2和34有关，3和4有关：

| | | | | | | |
—|—|—|—|—|—|—|—
| | 4| |4 |4 | | |
| | |3 | |3 | | |
|2 | |2 |2 | | | |
|1 |1 | | | | | |

D. Array Without Local Maximums（dp）

http://codeforces.com/contest/1068/problem/D

题意：有一个由[1,200]间的数组成的n个数的序列，这个序列满足下图条件，即对于第2个到第n-1个元素要求相邻元素至少有一个大于等于它，现在这个序列有一部分不记得了，不记得的位置标记为-1，求恢复的方法数

dp[i][j][k]表示第i个数字为j，且第i个数和第i-1个数的关系为k时的方案数，k取值为0，1，2分别代表小于等于大于

状态转移的时候对于某个不确定的位的方法数，可以用一个前缀和tmp累加，求小于时j从小到大，求大于时j从大到小

最后答案就是∑(dp[n][i][1]+dp[n][i][2])，枚举最后一位i[1,200]相加即可

E. Multihedgehog（dfs，找多叉树的根）

http://codeforces.com/contest/1068/problem/E

题意：

「k-刺猬」定义如下

1-刺猬：有一个度数至少为3的点和一些度数为1的点

k-刺猬（k>=2）：如果把所有(k-1)-刺猬看成一个度数为1的点，那么有一个度数至少为3的点和一些度数为1的点

给一棵树，问这棵树是不是k-刺猬

一种思路是模拟：找度数为1的点，然后同父亲且三个以上的合并，父亲在下一轮作为一个新的度数为1的结点，一直模拟下去判断是否符合定义即可

所以还有一种思路是dfs:先找到树根，然后从树根开始往下dfs判断是否是合法的k-刺猬，判断就是如果当前深度为1就判断度数是否为1，如果当前深度不为1就dfs儿子，如果儿子都合法判断儿子个数是否大于等于3

个人感觉。。第二种优美一些，就写的第二种
