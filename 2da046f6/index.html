3.2代码区

def on_start(self)是脚本的入口点，如果按run就会调用它

self.crawl('__START_URL__', callback=self.index_page)是这里最重要的API，添加一个爬取新任务，__START_URL__处输入要爬取的网站，通过self.crawl的参数进行详细设置

def index_page(self, response)得到一个Response对象，response.doc是一个pyquery对象，它具有类似jQuery的API来选择要提取的元素

def detail_page(self, response)返回一个dict对象作为结果，结果将默认被捕获到resultdb中
